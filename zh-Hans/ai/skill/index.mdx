技能是 Xpert 智能体可复用的能力包。一个技能可以为智能体提供一组聚焦的指令、辅助文件、Prompt Workflow、脚本、模板、素材或特定任务行为。团队可以把可重复使用的能力安装到工作区，再让不同智能体按需加载。

当某个能力需要反复使用，但还不需要开发成完整工具集或插件时，适合使用技能。例如代码审查技能、文档共创技能、演示文稿技能、内部运营流程技能等。

## 技能提供什么

- **可复用指令**：每个技能围绕一个 `SKILL.md` 文件展开，用于说明智能体何时以及如何使用该技能。
- **辅助资源**：技能可以包含脚本、参考资料、模板、素材或示例。
- **工作区安装**：安装后的技能归属于某个工作区，可在技能工作台中查看、编辑、上传或删除。
- **运行时启用**：智能体通过 Skills Middleware 加载选中的技能，也可以在 ChatKit 中让用户按需选择。
- **组织共享**：工作区上传的技能可以分享为组织市场中的技能，供其他工作区安装。
- **仓库发现**：管理员可以连接技能仓库，让团队从索引中搜索、安装和更新技能。

## 技能生命周期

1. 在技能市场或仓库索引中发现技能。
2. 将技能安装到目标工作区。
3. 在工作区技能工作台中检查和编辑技能文件。
4. 创建或配置智能体时启用所需技能。
5. 智能体运行时，Skills Middleware 会把选中的技能同步到沙箱，并把简要技能信息加入上下文。
6. 根据需要把工作区技能分享到组织市场，或由管理员统一维护技能仓库与模板同步。

## 在哪里使用技能

| 区域 | 可以做什么 |
| --- | --- |
| Explore / Skills | 浏览精选技能、搜索仓库索引、查看详情、安装或更新技能到工作区。 |
| Workspace / Skills | 管理已安装技能、从 GitHub 安装、上传 ZIP、注册仓库、查看和编辑技能文件。 |
| 智能体创建向导 | 创建空白智能体时选择技能，并自动接入 Skills Middleware。 |
| Agent Studio | 配置 Skills Middleware 默认技能、系统提示和技能命令。 |
| Settings / Tenant / Skills | 初始化公共技能仓库、同步模板技能、上传组织技能、管理技能仓库。 |
