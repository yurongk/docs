变更集是任务执行过程中的变更快照。每次同步都会生成新的变更集，便于按阶段审阅。

## 变更集包含内容

- 文件变更列表
- 统一/分栏 diff
- 新增与删除行统计
- 可选的 PR 信息

## Diff 视图

支持两种视图：

- 分栏 diff：左右对比
- 统一 diff：单流展示

可在任务页中随时切换。

## 复制辅助

在任务头部菜单中可以：

- 复制 patch
- 复制 git apply

便于本地应用或分享给团队成员。

## 审阅建议

- 优先查看最新变更集。
- 先聚焦高影响文件。
- 用增删行快速判断改动规模。
